signalk-parser-nmea0183
=======================

A parser written in node.js to parse NMEA 0183 sentences into Signal K. 


SUPPORTED NMEA SENTENCES
------------------------
- APB
- DBT
- GGA
- GLL
- MWV
- RMC
- VDM
- VDO
- VTG
